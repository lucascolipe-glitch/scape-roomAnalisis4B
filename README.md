## Página docente

Archivo: `docente.html`  
Clave: `FUNCIONES`

No tiene seguridad real porque GitHub Pages publica archivos estáticos. Si querés que el estudiantado no entre, cambiá el nombre del archivo antes de subirlo.

## Publicar en GitHub Pages

1. Crear un repositorio nuevo.
2. Subir todos los archivos de esta carpeta, dejando `index.html` en la raíz.
3. Ir a `Settings` -> `Pages`.
4. Elegir `Deploy from a branch`, rama `main`, carpeta `/root`.
5. Guardar y esperar la URL pública.

## Editar colores

Editar `css/estilos.css` en el bloque `:root`.

## Editar códigos

Editar `js/escape.js`, objeto `ESCAPE.codes`.
