Estación 4

Señal eléctrica

Una señal crece rápidamente y se acerca a un nivel mínimo.

Modelo exponencial

\[ p(x)=2^x-4 \]

Realicen el análisis completo.

  1. Dominio.
  2. Imagen.
  3. Raíz.
  4. Ordenada al origen.
  5. Asíntota horizontal.
  6. Crecimiento o decrecimiento.
  7. Positividad y negatividad.

Obtener el código

El código se forma con:

  • dos cifras para la raíz;
  • dos cifras para el valor absoluto de la asíntota horizontal.
Para encontrar la raíz, igualen a cero: \(2^x-4=0\). Recuerden que \(4=2^2\).
