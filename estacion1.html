Estación 1

Sensor de avance

El sistema detecta una magnitud que cambia siempre al mismo ritmo.

Modelo lineal

\[ f(x)=3x-6 \]

Analicen la función y registren la información en la hoja del grupo.

  1. Dominio.
  2. Raíz.
  3. Ordenada al origen.
  4. Pendiente.
  5. Crecimiento o decrecimiento.
  6. Positividad y negatividad.

Obtener el código

El código se forma con cuatro cifras:

  • dos cifras para la raíz;
  • dos cifras para el valor absoluto de la ordenada al origen.
Busquen primero dónde la función vale cero. Luego calculen cuánto vale cuando \(x=0\).
