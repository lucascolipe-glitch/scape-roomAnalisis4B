Estación 5

Escala logarítmica

El último módulo del tablero trabaja con una escala que solo acepta entradas positivas.

Modelo logarítmico

\[ q(x)=\log_3(x-2) \]

Realicen el análisis completo.

  1. Dominio.
  2. Imagen.
  3. Raíz.
  4. Ordenada al origen, si existe.
  5. Asíntota vertical.
  6. Crecimiento o decrecimiento.
  7. Positividad y negatividad.

Obtener el código

El código se forma con:

  • asíntota vertical;
  • raíz;
  • base del logaritmo;
  • 0, porque no hay ordenada al origen.
Para que exista el logaritmo, lo que está adentro debe ser positivo: \(x-2>0\). Para la raíz, usen que \(\log_3(1)=0\).
