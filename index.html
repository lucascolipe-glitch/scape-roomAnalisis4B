El banco de pruebas del taller quedó bloqueado. Para reactivarlo tendrán que recuperar cinco modelos matemáticos: función lineal, cuadrática, racional, exponencial y logarítmica.

Sistema bloqueado

Cada estación propone una función. Para avanzar no alcanza con reemplazar valores: hay que hacer un análisis completo, justificar las decisiones y obtener un código para abrir el candado físico.

En cada caja encontrarán una palabra. Al final, esas palabras ayudan a reactivar el sistema.

Reglas de juego

  • Trabajen en grupos de 3 o 4 integrantes.
  • Registren dominio, raíces, ordenada al origen, signo, crecimiento y asíntotas cuando corresponda.
  • El código digital debe coincidir con el candado físico.
  • Las pistas ayudan, pero primero intenten resolver con el grupo.
  • El docente puede pedir una justificación antes de abrir una caja.
